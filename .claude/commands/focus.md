Carica solo i file di un package specifico per review, fix o domande mirate.

**Package disponibili:** `home` | `drawer` | `dock` | `settings` | `theme` | `di` | `data` | `domain` | `navigation` | `widgets`

**Procedura:**
1. Mappa il package al path: `app/src/main/java/com/hina/minimalos/ui/<package>/` (o `di/`, `data/`, `domain/`)
2. Leggi tutti i file `.kt` in quel path — niente altro
3. Riporta struttura: file trovati + classi/composable principali
4. Aspetta istruzione su cosa fare con quei file

**Argomento ricevuto:** $ARGUMENTS
